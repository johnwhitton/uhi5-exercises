## Usage

### Initial Setup

```shell
# Clone this repository
git clone https://github.com/johnwhitton/uhi5-project.git
```

### Installation

```shell
# Install libraries includeing forge-std and v4-periphery
forge install
```

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
